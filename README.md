# Usage
```sh
pipenv install
pipenv run python main.py --help
```

```sh
usage: BigQuery Job Scheduler [-h] [--sql-directory SQL_DIRECTORY] project

Execute jobs and generate depencency graph.

positional arguments:
  project               Google Cloud project

options:
  -h, --help            show this help message and exit
  --sql-directory SQL_DIRECTORY
                        sql directory
```
